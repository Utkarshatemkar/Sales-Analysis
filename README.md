# 📊 Sales Analysis Project (Python + Power BI + Machine Learning)

## 🚀 Overview
This project is an end-to-end data analysis solution that uses Python for data processing and machine learning, and Power BI for interactive visualization.

It helps analyze sales performance, identify trends, and predict profit using a simple machine learning model.

---

## 🎯 Objectives
- Clean and process raw sales data
- Perform exploratory data analysis (EDA)
- Generate business insights
- Build a predictive model for profit estimation
- Create interactive dashboards in Power BI

---

## 🛠 Tools & Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Power BI
- Jupyter / VS Code
